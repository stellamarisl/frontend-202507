Curso de desarrollo Frontend

Este curso de desarrollo Frontend está diseñado para proporcionar a los estudiantes los conocimientos y habilidades necesarios para crear y mantener sitios web atractivos y funcionales. A lo largo del curso, los estudiantes aprenderán sobre las tecnologías fundamentales del desarrollo web, incluyendo HTML, CSS, Sass, JavaScript

  1. Introducción a HTML
  2. Introducción a CSS
  3. Introducción a Sass
  4. Introducción a JavaScript
  5. Introducción a React
